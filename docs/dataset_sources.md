## Dataset purpose
The dataset contains public information about Airbnb listings, hosts,
availability, prices, reviews, and neighbourhoods.

## Important limitation
Inside Airbnb data represents a scraped snapshot. Availability does not
directly confirm that a listing was booked. Reviews are also an incomplete
proxy for actual bookings.
